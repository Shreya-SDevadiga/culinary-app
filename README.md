# Culinary App 🍽️

A full-stack recipe sharing platform where users can create, share, and discover recipes. Includes admin panel for content moderation.

## Features
- 🔐 User authentication (register/login)
- 📝 Create, edit, and delete recipes
- ⭐ Rate and comment on recipes
- 🔖 Bookmark favourite recipes
- 👨‍💼 Admin dashboard for managing users, recipes & comments
- 📸 Image upload for recipes
- 🔍 Browse and search recipes

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB (local)
- **Auth:** JWT

## Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB installed locally

### 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/culinary-app.git
cd culinary-app

### 2. Setup Backend
cd backend
npm install
node server.js

### 3. Setup Frontend
cd frontend
npm install
npm start
